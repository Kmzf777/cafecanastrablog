## Story
**As a** Rafael (proprietário do Café Canastra),
**I want** um sistema de login simples para acessar o painel `/admin` com uma senha definida em variável de ambiente,
**so that** apenas eu possa criar, editar e excluir posts do blog, sem expor o painel ao público.

## Acceptance Criteria

1. Existe uma página de login em `/admin/login` com:
   - Campo de senha (input type="password")
   - Botão "Entrar"
   - Mensagem de erro para senha incorreta
   - Design compatível com o estilo do projeto (Tailwind CSS)

2. Ao inserir a senha correta (definida em `ADMIN_PASSWORD` no `.env`), o sistema:
   - Gera um token JWT assinado com `ADMIN_JWT_SECRET` (também do `.env`)
   - Define o token como cookie `admin_token` (httpOnly, secure, sameSite: lax)
   - Redireciona para `/admin`

3. O middleware (`middleware.ts`) protege todas as rotas `/admin/*` (exceto `/admin/login`):
   - Verifica a presença e validade do cookie `admin_token`
   - Se inválido ou ausente: redireciona para `/admin/login`
   - Se válido: permite o acesso

4. Existe uma rota de logout (`/api/admin/logout`) que:
   - Remove o cookie `admin_token`
   - Redireciona para `/admin/login`

5. Existe uma rota de API de login (`/api/admin/login`) que:
   - Recebe `{ password: string }` via POST
   - Compara com `ADMIN_PASSWORD` usando comparação de tempo constante (para evitar timing attacks)
   - Retorna 401 se incorreta, 200 com cookie se correta

6. O token JWT expira em 7 dias.

7. Nenhuma senha ou segredo é hardcoded no código — apenas lido de variáveis de ambiente.

8. O `build` e `lint` passam sem erros.

### Implementação do JWT com `jose`

```typescript
// Gerar token (em /api/admin/login)
import { SignJWT } from 'jose'

const secret = new TextEncoder().encode(process.env.ADMIN_JWT_SECRET)
const token = await new SignJWT({ role: 'admin' })
  .setProtectedHeader({ alg: 'HS256' })
  .setExpirationTime('7d')
  .setIssuedAt()
  .sign(secret)

// Verificar token (em middleware.ts)
import { jwtVerify } from 'jose'

const secret = new TextEncoder().encode(process.env.ADMIN_JWT_SECRET)
const { payload } = await jwtVerify(token, secret)
```

### Comparação Segura de Senha (Timing Attack)

```typescript
import crypto from 'crypto'

function safeCompare(a: string, b: string): boolean {
  // Garante mesmo comprimento antes de comparar
  if (a.length !== b.length) return false
  return crypto.timingSafeEqual(
    Buffer.from(a, 'utf8'),
    Buffer.from(b, 'utf8')
  )
}

// Uso:
const isValid = safeCompare(password, process.env.ADMIN_PASSWORD || '')
```

### Cookie HTTP-Only

```typescript
import { cookies } from 'next/headers'

// Setar cookie (server-side no route handler)
const cookieStore = cookies()
cookieStore.set('admin_token', token, {
  httpOnly: true,
  secure: process.env.NODE_ENV === 'production',
  sameSite: 'lax',
  maxAge: 60 * 60 * 24 * 7, // 7 dias em segundos
  path: '/',
})
```

### Middleware Next.js — Verificação JWT

```typescript
// middleware.ts (adicionar à lógica existente)
import { jwtVerify } from 'jose'
import { NextRequest, NextResponse } from 'next/server'

export async function middleware(request: NextRequest) {
  const pathname = request.nextUrl.pathname

  // Proteger rotas /admin (exceto /admin/login)
  if (pathname.startsWith('/admin') && !pathname.startsWith('/admin/login')) {
    const token = request.cookies.get('admin_token')?.value

    if (!token) {
      return NextResponse.redirect(new URL('/admin/login', request.url))
    }

    try {
      const secret = new TextEncoder().encode(process.env.ADMIN_JWT_SECRET)
      await jwtVerify(token, secret)
    } catch {
      return NextResponse.redirect(new URL('/admin/login', request.url))
    }
  }

  return NextResponse.next()
}

export const config = {
  matcher: ['/admin/:path*'],
}
```
