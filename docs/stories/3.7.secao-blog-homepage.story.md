## Story
**As a** visitante da homepage do Café Canastra,
**I want** ver uma prévia dos posts mais recentes do blog diretamente na homepage,
**so that** eu descubra o conteúdo editorial da marca e seja levado a explorar o blog, aumentando o tempo no site e o tráfego orgânico.

## Acceptance Criteria

1. Existe uma seção "Do Nosso Blog" na homepage (`/`) que exibe os 3 posts mais recentes com `status = 'published'`, ordenados por `published_at DESC`.

2. Cada card de post exibe:
   - Imagem de capa (com fallback para imagem padrão se não houver)
   - Categoria (badge)
   - Título do post
   - Excerpt (truncado em 120 chars se necessário)
   - Data de publicação formatada em pt-BR
   - Link "Ler mais →" para `/blog/[slug]`

3. A seção possui um botão/link "Ver todos os posts" que navega para `/blog`.

4. Quando não há nenhum post publicado, a seção **não é exibida** na homepage (sem mensagem de erro ou placeholder vazio).

5. A seção é visualmente consistente com o estilo existente da homepage (cores âmbar/marrom, tipografia do projeto, responsiva).

6. As imagens dos cards usam `<Image>` do Next.js com `loading="lazy"` e dimensões definidas.

7. A seção busca os posts no servidor (Server Component) para não aumentar o bundle JS do cliente.

8. O build e lint passam sem erros.

## Dev Notes

### Atenção: Homepage é "use client"

A homepage atual (`app/page.tsx`) usa `"use client"`, o que significa que ela é um Client Component. Para usar um Server Component dentro dela de forma eficiente, há duas abordagens:

**Opção A (recomendada):** Extrair a busca de posts para um Server Component separado e importá-lo:

```tsx
// components/BlogPreviewSection.tsx (Server Component — sem 'use client')
import { createServerClient } from '@/lib/supabase-server'
import { BlogPreviewCard } from './BlogPreviewCard'

export async function BlogPreviewSection() {
  const supabase = createServerClient()
  const { data: posts } = await supabase
    .from('blog_posts')
    .select('id, title, slug, excerpt, image_url, image_alt, category, published_at')
    .eq('status', 'published')
    .order('published_at', { ascending: false })
    .limit(3)

  if (!posts || posts.length === 0) return null

  return (
    <section className="py-16 px-4 bg-amber-50">
      <div className="max-w-6xl mx-auto">
        <h2 className="text-3xl font-playfair text-amber-900 text-center mb-12">
          Do Nosso Blog
        </h2>
        <div className="grid grid-cols-1 lg:grid-cols-3 gap-8">
          {posts.map(post => (
            <BlogPreviewCard key={post.id} post={post} />
          ))}
        </div>
        <div className="text-center mt-10">
          <a href="/blog" className="text-amber-700 hover:text-amber-900 font-medium">
            Ver todos os posts →
          </a>
        </div>
      </div>
    </section>
  )
}
```

**Opção B:** Buscar posts no cliente via `useEffect` na homepage existente (menos eficiente para SEO).

**Recomendação:** Usar Opção A. No Next.js App Router, Server Components podem ser importados dentro de Client Components — o Next.js gerencia a renderização corretamente.

### Fallback de Imagem

```tsx
// BlogPreviewCard.tsx
import Image from 'next/image'
import Link from 'next/link'

const FALLBACK_IMAGE = '/banner-cafecanastra.png'

export function BlogPreviewCard({ post }: { post: Partial<BlogPost> }) {
  const imageUrl = post.image_url || FALLBACK_IMAGE
  const imageAlt = post.image_alt || post.title || 'Post do blog Café Canastra'
  const excerpt = post.excerpt
    ? post.excerpt.slice(0, 120) + (post.excerpt.length > 120 ? '...' : '')
    : ''

  return (
    <article className="bg-white rounded-xl overflow-hidden shadow-sm hover:shadow-md transition-shadow">
      <Link href={`/blog/${post.slug}`}>
        <div className="relative h-48">
          <Image
            src={imageUrl}
            alt={imageAlt}
            fill
            className="object-cover"
            sizes="(max-width: 768px) 100vw, 33vw"
          />
        </div>
        <div className="p-5">
          {post.category && (
            <span className="text-xs text-amber-700 uppercase tracking-wide font-medium">
              {post.category}
            </span>
          )}
          <h3 className="text-lg font-semibold text-gray-900 mt-1 mb-2">{post.title}</h3>
          <p className="text-gray-600 text-sm">{excerpt}</p>
          <span className="text-amber-600 text-sm font-medium mt-3 inline-block">
            Ler mais →
          </span>
        </div>
      </Link>
    </article>
  )
}
```
