## Story
**As a** responsável pelo SEO do Café Canastra,
**I want** que o site gere automaticamente um `sitemap.xml` dinâmico que inclua todas as páginas estáticas e todos os posts do blog publicados,
**so that** o Google rastreie e indexe todas as URLs do site de forma eficiente, e novos posts apareçam no sitemap automaticamente ao serem publicados.

## Acceptance Criteria

1. O arquivo `sitemap.xml` é acessível em `https://[domínio]/sitemap.xml` e retorna XML válido.

2. O sitemap inclui as seguintes páginas estáticas com `priority` e `changefreq` adequados:
   - `/` — priority: 1.0, changefreq: weekly
   - `/blog` — priority: 0.9, changefreq: daily
   - `/sobre/origem` — priority: 0.7, changefreq: monthly

3. O sitemap inclui dinamicamente todos os posts do blog com `status = 'published'`, cada um com:
   - `<loc>`: URL absoluta do post (`https://[domínio]/blog/[slug]`)
   - `<lastmod>`: data de `updated_at` do post (formato ISO 8601)
   - `<priority>`: 0.8
   - `<changefreq>`: weekly

4. Quando um novo post é publicado (status muda para `published`), ele aparece automaticamente no sitemap na próxima requisição — sem necessidade de rebuild ou deploy.

5. O `robots.txt` referencia o sitemap: `Sitemap: https://[domínio]/sitemap.xml`.

6. O `robots.txt` bloqueia `/admin` e libera o restante do site.

7. A implementação usa a API nativa de sitemap do Next.js (`app/sitemap.ts`) — não bibliotecas externas.

8. O build (`npm run build`) passa sem erros TypeScript.

## Dev Notes

### API Nativa de Sitemap do Next.js App Router

```typescript
// app/sitemap.ts
import { MetadataRoute } from 'next'
import { createClient } from '@/lib/supabase-server' // cliente server-side

export const revalidate = 300 // revalidar a cada 5 minutos

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  const siteUrl = process.env.NEXT_PUBLIC_SITE_URL || 'https://cafecanastra.com'

  const supabase = createClient()
  const { data: posts } = await supabase
    .from('blog_posts')
    .select('slug, updated_at')
    .eq('status', 'published')

  const blogPosts: MetadataRoute.Sitemap = (posts || []).map((post) => ({
    url: `${siteUrl}/blog/${post.slug}`,
    lastModified: new Date(post.updated_at),
    changeFrequency: 'weekly',
    priority: 0.8,
  }))

  return [
    {
      url: siteUrl,
      lastModified: new Date(),
      changeFrequency: 'weekly',
      priority: 1.0,
    },
    {
      url: `${siteUrl}/blog`,
      lastModified: new Date(),
      changeFrequency: 'daily',
      priority: 0.9,
    },
    {
      url: `${siteUrl}/sobre/origem`,
      lastModified: new Date(),
      changeFrequency: 'monthly',
      priority: 0.7,
    },
    ...blogPosts,
  ]
}
```

### API Nativa de Robots.txt

```typescript
// app/robots.ts
import { MetadataRoute } from 'next'

export default function robots(): MetadataRoute.Robots {
  const siteUrl = process.env.NEXT_PUBLIC_SITE_URL || 'https://cafecanastra.com'
  return {
    rules: {
      userAgent: '*',
      allow: '/',
      disallow: ['/admin', '/admin/'],
    },
    sitemap: `${siteUrl}/sitemap.xml`,
  }
}
```
