## Story
**As a** responsável pelo SEO do Café Canastra,
**I want** que cada página do site tenha `<title>`, `<meta name="description">`, `<meta property="og:*">` e `<link rel="canonical">` únicos e otimizados,
**so that** o Google indexe corretamente cada página com o conteúdo relevante, evite conteúdo duplicado, e as páginas sejam compartilháveis com preview rico em redes sociais.

## Acceptance Criteria

1. A homepage (`/`) possui:
   - `<title>`: "Café Canastra | Café Especial da Serra da Canastra, MG"
   - `<meta name="description">`: texto descritivo com 150-160 caracteres incluindo a palavra-chave principal
   - `<meta property="og:title">`, `og:description`, `og:image`, `og:url`, `og:type`
   - `<link rel="canonical" href="https://[domínio]/">`

2. A página de listagem do blog (`/blog`) possui meta tags únicas referentes ao blog:
   - `<title>`: "Blog | Café Canastra — Dicas, Receitas e Curiosidades"
   - Meta description específica do blog
   - Canonical correto

3. Cada post individual do blog (`/blog/[slug]`) possui meta tags dinâmicas geradas a partir dos dados do post no banco:
   - `<title>`: `{título do post} | Café Canastra`
   - `<meta name="description">`: excerpt do post (máx 160 chars)
   - `og:image`: imagem de capa do post
   - `og:type`: "article"
   - Canonical: URL canônica do post

4. A página de origem (`/sobre/origem`) possui meta tags únicas sobre terroir e origem.

5. Nenhuma página pública possui `<meta name="robots" content="noindex">` não intencional.

6. A rota `/admin` e sub-rotas possuem `noindex, nofollow` para não ser indexada.

7. O `og:image` padrão (fallback) é a imagem `banner-cafecanastra.png` disponível em `/public/`.

8. A implementação usa o sistema `generateMetadata()` do Next.js App Router — sem bibliotecas externas.

## Dev Notes

### Sistema de Metadata do Next.js App Router
O Next.js 16+ usa a API `generateMetadata()` para metadata dinâmico e exportações `metadata` para estático.

```typescript
// Layout raiz — app/layout.tsx
import type { Metadata } from 'next'

export const metadata: Metadata = {
  metadataBase: new URL(process.env.NEXT_PUBLIC_SITE_URL || 'https://cafecanastra.com'),
  title: {
    template: '%s | Café Canastra',
    default: 'Café Canastra | Café Especial da Serra da Canastra, MG',
  },
  description: 'Café especial produzido na Serra da Canastra, Minas Gerais. Altitude, terroir único e torra artesanal para uma xícara excepcional.',
  openGraph: {
    siteName: 'Café Canastra',
    images: [{ url: '/banner-cafecanastra.png', width: 1200, height: 630 }],
  },
}

// Página dinâmica — app/blog/[slug]/page.tsx
export async function generateMetadata({ params }: { params: { slug: string } }): Promise<Metadata> {
  const post = await getPostBySlug(params.slug) // busca no Supabase
  return {
    title: post.title,
    description: post.excerpt,
    openGraph: {
      type: 'article',
      images: post.image_url ? [{ url: post.image_url }] : undefined,
      publishedTime: post.published_at,
    },
    alternates: {
      canonical: `/blog/${params.slug}`,
    },
  }
}
```
