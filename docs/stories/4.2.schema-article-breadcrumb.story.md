## Story
**As a** responsável pelo SEO e GEO do Café Canastra,
**I want** que cada post do blog tenha schema markup `Article` completo e que todas as páginas internas tenham `BreadcrumbList` em JSON-LD,
**so that** o Google exiba rich snippets de artigo nos resultados de busca e IAs generativas identifiquem os posts como conteúdo estruturado e citável.

## Acceptance Criteria

1. Cada post individual (`/blog/[slug]`) possui schema `Article` em JSON-LD com os campos:
   - `@type`: "Article"
   - `headline`: título do post (máx 110 chars para rich snippet)
   - `description`: excerpt do post
   - `image`: URL absoluta da imagem de capa (ou fallback)
   - `datePublished`: `published_at` em ISO 8601
   - `dateModified`: `updated_at` em ISO 8601
   - `author`: objeto `Organization` com nome "Café Canastra"
   - `publisher`: objeto `Organization` com nome e logo
   - `mainEntityOfPage`: URL canônica do post

2. A listagem do blog (`/blog`) possui schema `Blog` com:
   - `@type`: "Blog"
   - `name`: "Blog Café Canastra"
   - `description`: descrição do blog
   - `url`: URL da listagem
   - `publisher`: objeto `Organization`

3. Todas as páginas internas possuem `BreadcrumbList` correspondente ao breadcrumb visual:
   - Homepage (`/`): não precisa de breadcrumb
   - `/blog`: `Home > Blog`
   - `/blog/[slug]`: `Home > Blog > [Categoria] > [Título do Post]`
   - `/sobre/origem`: `Home > Sobre > Nossa Origem`
   - `/admin/*`: não precisa (noindex)

4. Todos os schemas validam sem erros no Google Rich Results Test.

5. Os schemas são implementados usando o componente `SchemaOrg` criado na Story 2.2 (ou equivalente).

6. Nenhum schema possui dados hardcoded desnecessários — todos os campos variáveis vêm dos dados do post ou das configurações centrais.

7. O build e lint passam sem erros.

## Dev Notes

### Schemas a Implementar

**Article (para cada post):**
```typescript
export function generateArticleSchema(post: BlogPost, siteUrl: string) {
  return {
    "@context": "https://schema.org",
    "@type": "Article",
    "headline": post.title.slice(0, 110),
    "description": post.excerpt || '',
    "image": post.image_url || `${siteUrl}/banner-cafecanastra.png`,
    "datePublished": post.published_at,
    "dateModified": post.updated_at,
    "author": {
      "@type": "Organization",
      "name": "Café Canastra",
      "url": siteUrl
    },
    "publisher": {
      "@type": "Organization",
      "name": "Café Canastra",
      "logo": {
        "@type": "ImageObject",
        "url": `${siteUrl}/logo-canastra.png`
      }
    },
    "mainEntityOfPage": {
      "@type": "WebPage",
      "@id": `${siteUrl}/blog/${post.slug}`
    }
  }
}
```

**BreadcrumbList:**
```typescript
export function generateBreadcrumbSchema(items: { name: string; url: string }[]) {
  return {
    "@context": "https://schema.org",
    "@type": "BreadcrumbList",
    "itemListElement": items.map((item, index) => ({
      "@type": "ListItem",
      "position": index + 1,
      "name": item.name,
      "item": item.url
    }))
  }
}

// Uso no post:
generateBreadcrumbSchema([
  { name: "Home", url: siteUrl },
  { name: "Blog", url: `${siteUrl}/blog` },
  { name: post.category || "Artigo", url: `${siteUrl}/blog?category=${post.category}` },
  { name: post.title, url: `${siteUrl}/blog/${post.slug}` },
])
```

**Blog:**
```typescript
export function generateBlogSchema(siteUrl: string) {
  return {
    "@context": "https://schema.org",
    "@type": "Blog",
    "name": "Blog Café Canastra",
    "description": "Dicas, receitas e curiosidades sobre café especial da Serra da Canastra",
    "url": `${siteUrl}/blog`,
    "publisher": {
      "@type": "Organization",
      "name": "Café Canastra",
      "url": siteUrl
    }
  }
}
```
