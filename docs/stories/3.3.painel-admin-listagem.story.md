## Story
**As a** Rafael (proprietário do Café Canastra),
**I want** um painel admin em `/admin` que exibe todos os posts do blog em uma tabela com filtros por status e ações de criar, editar e excluir,
**so that** eu consiga gerenciar o calendário editorial do blog de forma visual e organizada.

## Acceptance Criteria

1. A página `/admin` (protegida por autenticação da Story 3.2) exibe uma tabela com todos os posts, com as colunas:
   - **Título** (truncado em 60 chars se necessário)
   - **Categoria**
   - **Status** (badge colorido: cinza=rascunho, amarelo=agendado, verde=publicado)
   - **Data Agendada** (formato: "01/03/2026 às 10:00" ou "—" se não agendado)
   - **Ações**: botões Editar e Excluir

2. Existe um filtro por status no topo da tabela com os botões:
   - "Todos" | "Rascunhos" | "Agendados" | "Publicados"
   - O filtro ativo é destacado visualmente
   - Filtrar atualiza a listagem sem recarregar a página (client-side)

3. Existe um botão "Novo Post" proeminente que navega para `/admin/posts/new`.

4. O botão Excluir exibe um modal/diálogo de confirmação antes de deletar:
   - Texto: "Tem certeza que deseja excluir o post '[título]'? Esta ação não pode ser desfeita."
   - Botões: "Cancelar" e "Excluir" (botão destrutivo em vermelho)

5. Após confirmar exclusão, o post é deletado via API e a listagem é atualizada sem recarregar a página.

6. Quando não há posts no filtro selecionado, exibe mensagem: "Nenhum post encontrado." com link para criar o primeiro.

7. A tabela exibe os posts ordenados por `created_at DESC` (mais recentes primeiro).

8. A página possui estado de loading enquanto carrega os posts.

9. Existe indicação visual da contagem de posts por status (ex: "3 rascunhos, 2 agendados, 5 publicados").

10. O build e lint passam sem erros.

### Estrutura de Componente Sugerida

```tsx
// app/admin/page.tsx
'use client'

import { useEffect, useState } from 'react'
import { BlogPost } from '@/lib/types/blog'
import { Button } from '@/components/ui/button'
import { toast } from 'sonner'

type FilterStatus = 'all' | 'draft' | 'scheduled' | 'published'

export default function AdminPage() {
  const [posts, setPosts] = useState<BlogPost[]>([])
  const [filter, setFilter] = useState<FilterStatus>('all')
  const [loading, setLoading] = useState(true)

  useEffect(() => {
    fetch('/api/admin/posts')
      .then(r => r.json())
      .then(data => { setPosts(data); setLoading(false) })
  }, [])

  const filtered = filter === 'all'
    ? posts
    : posts.filter(p => p.status === filter)

  const handleDelete = async (id: string, title: string) => {
    const res = await fetch(`/api/admin/posts/${id}`, { method: 'DELETE' })
    if (res.ok) {
      setPosts(prev => prev.filter(p => p.id !== id))
      toast.success(`Post "${title}" excluído.`)
    } else {
      toast.error('Erro ao excluir post.')
    }
  }

  // ... render
}
```

### API Route — Verificação de Autenticação

```typescript
// app/api/admin/posts/route.ts
import { cookies } from 'next/headers'
import { jwtVerify } from 'jose'

async function verifyAdminAuth() {
  const token = cookies().get('admin_token')?.value
  if (!token) throw new Error('Unauthorized')
  const secret = new TextEncoder().encode(process.env.ADMIN_JWT_SECRET)
  await jwtVerify(token, secret)
}

export async function GET() {
  try {
    await verifyAdminAuth()
    // ... buscar posts
  } catch {
    return Response.json({ error: 'Unauthorized' }, { status: 401 })
  }
}
```
